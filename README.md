# ♠ ♥ 撲克牌對戰遊戲 ♦ ♣

一款使用 Python 開發的終端機撲克牌對戰遊戲，支援單人對電腦及雙人對戰模式。

## 📁 檔案說明

| 檔案 | 說明 |
|------|------|
| `final_term1.py` | 玩家 vs 電腦模式（單人遊戲） |
| `final_term2.py` | 玩家 vs 玩家模式（雙人遊戲，含隱藏手牌機制） |
| `final_term3.py` | **主程式**（整合版，可選擇遊戲模式） |

## 🎮 遊戲規則

### 基本設定
- 雙方初始 HP：**25**
- 每人手牌：**5 張**
- 使用一副標準 **52 張**撲克牌

### 花色效果
| 花色 | 效果 |
|------|------|
| ♠ 黑桃 | 攻擊對方，造成牌面點數的傷害 |
| ♣ 梅花 | 攻擊對方，造成牌面點數的傷害 |
| ♥ 紅心 | 回復自己 HP，回復量為牌面點數 |
| ♦ 方塊 | 反擊效果，當對方出攻擊牌時，將傷害反彈給對方 |

### 點數對照
| 牌面 | 點數 |
|------|------|
| A | 1 |
| 2-10 | 對應數字 |
| J | 11 |
| Q | 12 |
| K | 13 |

## 🚀 如何執行

### 執行主程式（推薦）
```bash
python final_term3.py
```
執行後會出現選單，可選擇：
1. 玩家 vs 電腦
2. 玩家 vs 玩家
3. 離開遊戲

### 單獨執行各模式
```bash
# 玩家 vs 電腦
python final_term1.py

# 玩家 vs 玩家
python final_term2.py
```

## 🎯 遊戲流程

1. 遊戲開始，雙方各抽 5 張手牌
2. 每回合雙方各出一張牌
3. 根據花色結算效果（回血、攻擊、反擊）
4. 出牌後自動從牌庫補牌至 5 張
5. 任一方 HP ≤ 0 或牌庫用完時遊戲結束
6. HP 較高者獲勝

## 🤖 電腦 AI 策略

電腦會根據以下策略選牌：
1. 當 HP ≤ 10 時，優先出紅心回血
2. 否則優先出高攻擊力的黑桃/梅花
3. 其次選擇出方塊進行反擊
4. 最後才出紅心

## 💡 操作說明

- 出牌時輸入 **1-5** 的數字選擇手牌
- 按 **Enter** 進入下一回合
- 雙人模式中，玩家輪替時需按 Enter 切換，避免看到對方手牌

## 📋 系統需求

- Python 3.x
- 支援 Windows / Mac / Linux
